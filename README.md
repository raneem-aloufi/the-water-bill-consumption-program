# Water Bill Consumption Program 

A C++ application to monitor water usage and manage fines for high consumption.

## Features
* **Threshold Check:** Detects if the bill exceeds 500.
* **Fine System:** Applies a 200-unit fine for high usage.
* **Warning Logic:** Provides 3 attempts to pay the fine if the initial payment is insufficient.
* **Summary Table:** Displays a final bill summary using arrays.

## How to Run
1. Open any C++ compiler (like Code::Blocks or VS Code).
2. Copy and paste the code from `main.cpp`.
3. Compile and Run.
